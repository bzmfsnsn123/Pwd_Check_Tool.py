# Pwd_Check_Tool.py 密码强度检测工具
## 项目文件结构
pwd-project/
├── README.md       # 项目说明文档（本文件）
├── pwd_check.py    # 主程序代码
└── record.txt      # 自动生成，存储密码检测记录

## 使用说明
运行程序后，根据菜单数字选择对应功能操作。

## 数据存储规则
1. 本地自动生成 `record.txt` 文件保存所有密码检测记录
2. 文件记录每次检测的密码组成、长度、安全评级
3. 程序支持读取历史检测记录；每次检测自动追加写入新记录

## 核心新学习知识点
- 字符串成员判断：识别大小写字母、数字、特殊符号
- 三元表达式简化输出：`{"有" if 条件 else "无"}`
- 多维度打分算法，自动划分密码安全等级
- `for ch in pwd`：字符串逐字符遍历判断
- 文件 `a` 追加模式，保留全部历史记录，不覆盖旧内容
- getpass 模块隐藏密码输入，模拟软件安全输入框
- 多层if多条件组合判断、分级业务逻辑
- 完整异常捕获 + 空输入校验，避免程序崩溃
